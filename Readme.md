# AI Gen Project

> Generate original quotes, rhymes, x`lines—all fresh, AI-powered, and creative. No clichés, all bangers.

## Features

- **Original Quotes**: Inspirational, thought-provoking, or funny quotes, 100% AI-generated.  
- **Catchy Rhymes**: Short rhyming couplets or mini-poems with a playful, TikTok/Naija vibe.  
- **Flirty Pick-up Lines**: Clever, cheeky, and funny lines that feel fresh and original.  
- **Customizable Prompts**: Easily tweak the style, tone, and creativity of outputs.

## Tech Stack

- NextJs 
- OpenAI API (or your preferred AI engine)  

## Installation

1. Clone the repo:  
```bash
git clone https://github.com/Wealthometer
```
2. Install dependencies:
```bash
npm install
```
3. Set your AI API key in .env (example):
``` .env
npm install
```
4. Run the project:
```bash
npm run dev
```
